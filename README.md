# Naive-bayes-sex-classify

## Description
Judge the person's sex, boy or girl??

## Requirement
・Python 3.6.5 </br>
・scikit-learn 0.19.1

## Installation
```
$ git clone github.com/tatatakky/Naive-bayes-sex-classify.git
```

## Usage
```
$ python naive_bayes.py <someone's name>
```
It will output boy or girl.
